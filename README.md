## ⚡ Key Innovations & Project Showcase

### 🔍 [QuantVision](https://quant-vision.vercel.app)
A mission-critical financial analysis platform using **Llama 3.1** and **ChromaDB** for real-time market insights. Features a professional backtesting pipeline that evaluates cumulative returns vs buy-and-hold strategies with high precision.

### 🚗 [Vtrack: Traffic Analysis System](https://github.com/Anish0104/Vtrack-Traffic_Analysis_System)
Multi-class object detection pipeline built with **YOLOv8** and **ByteTrack**, achieving 25-30 FPS with real-time speed estimation and a sleek 3D dashboard integration.

### 🎓 [SkillGap AI](https://skillgapai.vercel.app/)
An AI career simulator utilizing **Gemini 2.5 Flash** for multi-turn technical interviews and structured performance evaluation based on resume-job gaps.

### 📄 [DocPilot](https://docpilot-ai.streamlit.app/)
RAG-powered pipeline for millisecond-latency Q&A over 500+ documentation pages, minimizing hallucinations through advanced vector search.
