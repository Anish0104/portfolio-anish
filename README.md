# 💎 Anish Shirodkar | AI/ML Research Engineer
### High-Performance Minimalist Portfolio

A professional, immersive **Minimalist Portfolio** designed for an AI/ML Research Engineer. This project prioritizes ultra-smooth performance, clean glassmorphism, and high-fidelity typography.

![Anish's Portfolio](https://anish-shirodkar.vercel.app/og-image.png)

## 📡 Core Mission Systems

- **Premium Minimalist UI:** A clean, silent luxury aesthetic featuring slate-blue accents and refined glassmorphism.
- **High-Performance Grid:** Replaced heavy 3D components with a responsive, performant grid for technical expertise and experience.
- **Smooth Interaction Layer:** Optimized magnetic buttons and spotlight effects for zero-lag interaction across all devices.
- **Bento Matrix Layout:** Structured project and bio sections with real-time EST clock synchronization.

## 🛠️ Tech Stack

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Animation:** [Framer Motion](https://www.framer.com/motion/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & Vanilla CSS custom tokens
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** Vercel Edge

## 🚀 Setup
```bash
npm install
npm run dev
```
